    校外租屋 #

    吳紀葶、林青億協編
    最後更新日期2021.03.15

    台灣學生校外租屋大多落於North Oakland、Shadyside、Squirrel Hill，為本篇主要介紹。每個人對於房屋的喜好不同，推薦「好」房子是極為困難的事情，列出自己的喜好（例如預算、距離、採光、地板等等）會比較容易尋找適合的房子。

    看屋時機與徵室友 #

    通常4月多開始會有人陸續釋出空位，5月多通常是房源最多的時候，過了5月房源會漸漸減少。雖然現場看房子比較有踏實感，但也是需要考量房源的多寡。找室友可以利用PTT、TSS Facebook社團、中國留學生組織CSSA或加入租屋微信群。

    租屋查詢 #

    物業管理公司通常有專屬的網站，提供數間公寓出租；私人租屋則通常是到租屋網站或微信群。
