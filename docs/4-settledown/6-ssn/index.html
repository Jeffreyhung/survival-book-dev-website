    申辦SSN / SSA Ineligibility Letter #

    最後更新日期2020.08.03

    申辦SSN #

    社會安全碼（Social Security Number, SSN）是一組9位數字號碼（XXX – XX – XXXX），由社會安全局（Social Security Administration, SSA）提供。 除了美國公民外，綠卡或臨時工作的居民均須申辦（一般無工作留學生是不能申請1），主要是為個人報稅、信用紀錄查詢（如申辦信用卡、貸款）、社會安全福利等的依據。 功能類似台灣身份證字號的，因此不要任意透露SSN，也盡量避免使用Email發送SSN。假冒、盜用SSN是聯邦重罪。

    註1：F1簽證學生可由校內工作、CPT或OPT獲得SSN，參照工作章節。

    申辦流程 #

    流程參照CMU、Pitt提供的官方文件。 去SSA前，SEVIS須要開通，抵達美國至少10天， 須得到學校提供的Social Security Letter（校內工作）或Work Eligibility Document（例如：CPT I-20、OPT I-20、AT DS-2019）。 在SSA提交申請後約7個工作天經由郵寄收到。

    申辦SSA Ineligibility Letter #

    無資格申請SSN的人可申請此文件用於申請駕照、State ID等。此文件不需要從學校得到文件許可即可自行申請。

    申辦流程 #

    攜帶護照（含簽證）、I-94前往SSA，當場會提供一封證明文書。

    SSA地點 #

    學校附近有兩個可供申請SSN的辦公室。 一個在Downtown、一個在East Liberty，都是公車可以便捷到達的地方。 很多人建議到East Liberty，除認為這地方的服務人員相較客氣、另外就是地點較近。 辦公時間為9:00 AM~4:00 PM，但至遲3:00 PM前就得抵達。
