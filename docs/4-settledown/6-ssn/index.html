    申辦SSN / SSN Denial Letter #

    最後更新日期2020.08.03

    社會安全碼（Social Security Number, SSN）是一組9位數字號碼（XXX – XX – XXXX）， 由社會安全局（Social Security Administration, SSA）免費提供。 除了美國公民外，綠卡或臨時工作的居民1）均須申辦， 主要是為個人報稅、信用紀錄查詢（如申辦信用卡、貸款）、社會安全福利等。 功能類似台灣身份證字號，非必要不要任意提供SSN或攜帶SSN卡，也盡量避免使用Email發送SSN。

    無工作的留學生一般是不能申請SSN，可申請SSN Denial Letter代替用於申請駕照或State ID等。

    無論申請SSN或SSN Denial Letter， 均需要參加完OIS/OIE Orientation和Immigration Check-In Session， 並開通SEVIS以及抵達美國至少10天。

    註1：F1簽證學生可由校內工作、CPT或OPT獲得SSN，參照工作章節。

    申辦SSN #

    須得到學校提供的Social Security Letter（校內工作）或Work Eligibility Document（例如：CPT I-20、OPT I-20、AT DS-2019）。

    疫情前申辦流程
    流程參照CMU、Pitt提供的官方文件。主要流程：

    1. 準備護照、I-94、I-20、Social Security Letter、EAD (if on OPT)等文件。
    2. 親自前往SSA提交申請。
    3. SSN卡約7個工作天會郵寄收到住處。

    疫情間申辦流程
    由於每間SSA有不同的申辦流程，建議先致電欲前往的SSA詢問流程。參考流程：

    1. 寄送文件（I-20、簽證等重要文件為影本即可）。
    2. SSA會連絡至現場辦理。
    3. SSN卡會郵寄至住處。

    申辦SSN Denial Letter #

    此文件不需要從學校得到文件許可即可自行申請。

    疫情前申辦流程
    親自前往SSA提交申請後，攜帶護照（含簽證）、I-94，當場會提供一封證明文書。

    疫情間申辦流程
    以下流程適用於匹茲堡地區：

    1. 填寫Form SS-5，並額外寫一份簡單說明,如"requesting an SSN denial letter for a Pennsylvania driver’s license"。
    2. 護照資料頁（即照片頁）以及簽證頁影本
    3. 將文件郵寄至 6117 Station St., Pittsburgh, PA 15206
    4. SSN Denial Letter會郵寄至住處。

    SSA地點 #

    學校附近有兩個可供申請SSN的辦公室。 一個在Downtown、一個在East Liberty，都是公車可以便捷到達的地方。 很多人建議到East Liberty，除認為這地方的服務人員相較客氣、另外就是地點較近。 辦公時間為9:00 AM~4:00 PM，但至遲3:00 PM前就得抵達。
