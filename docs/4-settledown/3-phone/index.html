    申辦手機門號 #

    最後更新日期2020.08.03

    美國的手機方案通常包含無限通話和簡訊。主要是以網路流量、流速區分價錢。

    名詞解釋 #

    • Prepaid（Pay as you go）：類似預付卡，但可以設定每月自動扣繳（可享$5~10折扣），也可以隨時改方案。
    • Postpaid：類似台灣的簽約方案，通常比較貴，主要用於買手機折扣。辦Postpaid通常須要SSN，如果沒有則以押金取代。
    • Individual Plan：一個帳戶只辦一個門號，講究個人隱私或覺得合辦套餐麻煩者適用。
    • Family Plan：一個帳戶有多個門號，不限於家人，可以與任何人組，通常越多人會有越多折扣。由一個人付錢，再向其他成員收取費用。進入或退出須要管理者操作。有些Family plan是共享流量的。辦Family plan通常須要SSN。
    • Unlimited Plan：即吃到飽。有低速版與高速版吃到飽。低速版是1.5Mbps（480p Streaming）；高速版是8Mbps或更快的全速4G LTE。
    • 通訊技術：美國主流通訊技術為兩種：1. CDMA、2. GSM，台灣帶來的手機由於訊號頻帶不同，可能其中或都不適用。如果沒準備換手機，最好到店裡試過再辦。

    三大電信公司 #

    Verizon、AT&T、T-Mobile1是美國三大主流電信商，到處都有實體店面2,3，以下為總體評價：

    平均訊號滿意度123
    訊號覆蓋率123
    費用（貴）123
    通訊技術CDMA
    		GSMGSM

    註1：Sprint於2020年4月併入T-Mobile。
    註2：實體店面有直營店與授權店之分。授權店往往有額外費用。
    註3：網路辦理通常會有最好的優惠。

    電信運營商 #

    電信運營商是租用四大電信公司中的網路，再自行包裝成方案販售。通常訊號強度、流速、覆蓋率只會低於採用的電信公司。以下是常見的電信運營商：

    電信公司特色
    Cricket：採用AT&T，其中的Unlimited Family Plan（$25/月，4~5人，480p Streaming）有實體店面。
    Google Fi：採用T-Mobile。費率採流量計費（全球均一價），也有Unlimited Plan與Family Plan。支援全球200+國家網路、電話，國際電話、簡訊費率便宜，適合經常出國的人使用。可以申請多張Data-only SIM卡，給不同人使用。
    Mint：採用T-Mobile。相當便宜，訊號覆蓋率偏低、高速行駛或人多地方通話品質、網速較差。
    CTExcel：採用T-Mobile，中國電信推出，支援中文服務，無限20國國際通話與無限全球簡訊，支援台灣手機頻帶。
    CUniq：採用T-Mobile，中國聯通推出，支援中文服務。可以在台灣網路上直接購買到，支援台灣手機頻帶。

    攜號轉網 #

    如果想更換運營商並保留原本門號，可以跟原運營商索取門號的Account Number和Pin Number，交給想更換的運營商轉網。
