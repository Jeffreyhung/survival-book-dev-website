    學校宿舍 #

    最後更新日期2020.05.15

    校園宿舍 #

    校園宿舍（On-campus）通常僅提供大學生住宿、提供基本床具、書桌並包含餐點購買計畫。

    Pitt：僅提供大學生住宿，包含家具，費用納入學費中（費用已含Utilities）。大一強制住宿，大二以上基本上都需要抽籤排隊（一人排到，同隊室友可以一起入住）。部分樓只提供給大一住宿，部分樓只收大二以上。大部分樓都是多人一間（沒有獨立房間）。強制購買Dining Pass＋Dining Dollars，大一限定購買至少Tier 2以上的（2020年價格為$2050 /學期）。建議選最少次的Dining Pass即可。

    CMU：包含家具，費用納入學費中（費用已含Utilities）。大一強制住宿，如果需要指定特定室友，須提供他人的Andrew ID。大二以上、研究所以上需要排隊。大一強制購買Traditional Dining Plan。

    學校公寓 #

    學校公寓（University-Owned）為大學擁有公寓，通常不提供家具。

    Pitt：大學生、研究生皆可入住，均不帶家具，租金採月收。 College Gardens（大學生、研究生、教職員）、 Darragh Street（研究生）、 Hyacinth Place（大學生、研究生）、 Mayflower Apartments（研究生）。

    CMU：CMU未提供研究生用的學校公寓，僅提供Off-Campus Housing網站查詢校外租屋。
