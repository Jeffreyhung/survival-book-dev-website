    短期旅遊平安險/留學險 #

    最後更新日期2020.05.15

    即使學校有提供保險，大多是從8~9月開始生效。 因此，在美國會有沒有醫療保險的空窗期。 離開台灣前，可以先選購短期的旅遊平安險或留學險（均須在出國前購買）。 由於美國的醫療費昂貴，所以記得買醫療給付較高類型的保險。
