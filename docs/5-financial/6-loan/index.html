    留學貸款 #

    最後更新日期2020.08.03

    政府籌辦留學生貸款均有優惠利率，如果資金仍有不足，也可以考慮銀行自辦的留學貸款。

    臺北市青年留學生就學貸款 #

    承貸銀行：台北富邦銀行，詳細規定參照該銀行網頁。

    資格：設籍台北市1年以上且高中、大學畢業於台灣者可以申請台北市留學貸款。領有台灣政府提供之各項公費或留學獎助學金者、貸有教育部留學生貸款、就學貸款不得申請。

    貸款金額：修讀碩士學位可以申貸100萬台幣，修讀博士學位可以申貸200萬台幣。

    利息補助：依納稅義務人（獨立申報或受父母扶養）的綜合所得淨額稅率，最高補助100%利息，補助10年。

    教育部留學貸款 #

    承貸銀行：多家銀行承貸，詳細規定參照教育部留學貸款專區網頁。

    資格：高中、大學畢業於台灣的國民可以申請教育部留學貸款。領有台灣政府提供之各項公費或留學獎助學金者或貸有教育部就學貸款不得申請。

    貸款金額：修讀碩士學位可以申貸100萬台幣，修讀博士學位可以申貸200萬台幣。

    利息補助：依家庭年收入（單者者與父母合計；結婚者與配偶合計），最高補助寬限期內（3~5年）100%利息。

    銀行自辦留學貸款 #

    承貸銀行：中國信託、合作金庫、台北富邦、玉山銀行、兆豐銀行、臺灣銀行等。

    資格：基本上僅須要國外大學院校研究所入學許可證明或在學證明即可。不與政府貸款、獎學金等補助有任何衝突。

    貸款金額：依各銀行有所不同。

    利息：依各銀行有所不同。
