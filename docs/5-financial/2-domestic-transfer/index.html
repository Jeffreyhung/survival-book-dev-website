    美國境內轉帳 #

    最後更新日期2020.09.15

    美國境內轉帳系統與台灣相差甚大，具有不同的轉帳方式，到帳時間、手續費、轉帳金額均有不同的規定。

    ACH Transfer #

    主要用於同一個人的不同銀行帳戶之間轉帳1，部分銀行支持同一間銀行不同人之間轉帳。 ACH Transfer在銀行網頁或手機App上被簡稱為Transfer或Money Transfer。 第一次個人轉帳前須要認證帳戶是自己並綁定2,3，類似台灣的約定帳戶。

    優點：通常是免費的，適合用於同一人帳戶之間的資金調動。
    缺點：轉帳通常需要2~3個工作日。轉出金額有每日或每月上限。

    註1：C2P（例如：薪資發放）與P2C（例如：繳納學費、房租）通常也是使用ACH Transfer。因此也會需要提供Routing Number和Account Number。
    註2：認證方式有兩種：1. 要求直接登入該銀行帳戶2. 銀行會臨時存些金額，要填寫出該數字。均可在網路上完成。
    註3：ACH綁定是不互通的，在A銀行綁定B銀行後，可以從A銀行推錢（ACH Push）至B銀行或從A銀行發起從B銀行拉錢（ACH Pull）。然而，並不能在B銀行對A銀行發起推錢或拉錢，須要在B銀行綁定A銀行才能進行。

    Wire Transfer #

    即電匯，適用於轉帳金額較大或時間急迫。 需要收款人的姓名、地址、Routing Number和Account Number。

    優點：當天到帳、可跨行、接受較大額轉帳。
    缺點：匯款方與收款方都有手續費（$15~20）。

    Zelle #

    Zelle是銀行間的即時轉帳系統，只須知道收款人的手機號碼或Email即可以轉帳。 如果開戶銀行已經是Zelle的合作夥伴（大銀行通常都是），銀行App通常已經內建Zelle。

    優點：即時到帳、可跨行、無手續費、操作方便。
    缺點：每日或每月轉出金額上限（例如：PNC $1000/日、$5000/月；Chase $2000/日、$16,000/月），且收款方需要有Zelle。目前只和美國本土銀行有合作。

    PayPal / Venmo #

    PayPal與Venmo均屬於第三方轉帳系統，付款是投過帳戶餘額或綁定的個人金融帳戶（支票帳戶、簽帳金融卡及信用卡）， 收款方則是即時存入系統帳戶裡，帳戶餘額需要自行轉入綁定的銀行帳戶。 由於是即時轉帳，一旦提交是不可以撤回，交易平台並不會為付款方提供保護，因此需要注意匯款金額與對象。

    PayPal：老牌的第三方轉帳軟體，兼具網路購物付款渠道。轉帳方面，透過Email尋找轉錢對象，使用簽帳金融卡和信用卡轉帳均需要手續費。 網路購物方面，買方的付款方式均無手續費，且具有保護信用卡信息的功能。經姓名認證後，單筆支付上限為$10,000，總額沒有限制。支援25種貨幣。

    Venmo：具社交元素的新興轉帳軟體，申辦僅須要姓名、Email、美國電話號碼即可。 透過帳號或QR-code找到轉錢對象，亦可以申請好友（非必須）。 除了信用卡轉帳是需要手續費外，其餘方式均為免費。 每周有轉出金額上限，未驗證帳號為$299.99、驗證帳號為$2,999.99。僅支援美元。

    Personal Check / Money Order / Cashier’s Check #

    Personal Check、Money Order、Cashier’s Check是美國常見的票據，三者用途、簽發方式不同。

    Personal Check：由支票帳戶（Checking Account）的Routing Number和Account Number所構成的付款票據（圖5-1）。 帳戶持有人填寫金額、簽字，即可用於付款。填寫方式可以參照網路或請教Banker。個人支票可向銀行、支票訂製公司或倉儲式大賣場（Costco、Walmart）購買，而部分銀行會免費提供高級帳戶免費的個人支票。個人支票不等於現金，收取要注意有跳票、撤銷、簽名不符等風險。兌現須要有銀行帳戶，除臨櫃外，可使用ATM或手機（有上限）。

    圖5-1 個人支票示意圖

    Money Order：保證兌換的票據，票面包含收款人（必要）、付款人（非必要）、雙方地址（非必要），金額上限為$1,000（圖5-2）。 可在郵局（USPS）、商店（Walmart、CVS）等地方購買，手續費約~$1。

    圖5-2 Money Order示意圖

    Cashier’s Check：銀行簽發的保證兌換票據，票面包含簽發銀行及其行員簽名、收款人資訊（圖5-3）， 購買是經由個人銀行帳戶扣款，通常用於大額購買（例如：購車）。手續費約$10~20。
