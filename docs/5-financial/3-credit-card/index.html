    信用卡 #

    最後更新日期2021.05.01

    信用卡使用基本守則：僅當成是支付工具，而非借款工具，以不延遲繳款、不預借現金、不分期付款為原則。本節主要簡略涉及使用信用卡優勢與福利，並介紹累積信用紀錄後的熱門信用卡。零信用紀錄信用卡申請請參照申辦信用卡章節。

    信用卡優勢 #

    信用卡（Credit Card）有許多簽帳金融卡（Debit Card）沒有的優勢。

    A. 安全性 #

    保障： 相較Debit Card，Credit Card美國法律對於盜刷（Unauthorized Purchases）有較全面保障。

    Credit CardDebit Card
    盜刷責任 (Liability limit)
    盜刷責任上限為$50。基於商業考量，銀行大多直接免除盜刷責任，即Zero liability。2天內的盜刷責任上限為$50、2~60天內為$500、60天以上全責。
    盜刷調查
    在調查期間，持卡人無須負擔盜刷金額。在調查期間，持卡人盜刷金額凍結，有資金週轉等風險。
    爭議款（Dispute）
    交易遇到交易糾紛時（例如：物品未寄達、收款金額錯誤等）， 有權向銀行提出爭議款（通常透過網路提交即可）。銀行對於Debit Card爭議款處理流程通常較為繁瑣與冗長。
    交易方式
    交易時不需輸入PIN碼，快速、方便。需要輸入PIN碼， 如果PIN碼遺失所造成的盜刷，銀行可拒絕理賠。

    B. 消費回饋 #

    開卡獎勵：申請信用卡大多提供獎勵，依卡種而有所不同，其價值約$0~$1,200。

    定期回饋：以現金回饋來說，最少有1%，多則4~10%。有些則以銀行點數、飯店點數、航空哩程等回饋。

    不定期回饋、折扣：2020年疫情期間，多家信用卡提供高額的餐廳、超商消費回饋。此外，Amex、Chase、BOA等發卡銀行會提供商家折扣，形式以打折或消費滿多少折多少為主。

    C. 福利 #

    信用卡提供延長保固、購物保險、租車保險，高階卡亦旅遊險、免費機場貴賓室等。所有福利均有其購買前提、理賠上限等，不同發卡機構、信用卡均有所不同，詳情須參閱各信用卡條款。

    初中階信用卡常見福利：

    延長保固（Extended Warranty）
    持卡人透過信用卡全額購買具有保固的商品時，發卡機構會提供額外1倍保固時間（一般最長不超過1年）。在額外延保期內，滿足條件的修理費用由發卡機構承擔。

    購物保障（Purchase Protection）
    持卡人透過信用卡購買的商品在一定時間內損壞（含意外損壞）或盜竊的話，可獲得理賠。Amex部分信用卡甚至還提供了遺失的理賠保障。

    租車保險（Rental Coverage）
    持卡人透過信用卡全額租用傳統租車公司（Car Sharing不符合）可獲得免費的Collision/Loss Damage Insurance，信用卡租車保險詳情參照租車章節。

    手機保險（Cellphone Protection）
    持卡人透過信用卡每月繳交手機電話費，發卡機構會自動為手機投保，投保的內容包括損壞和盜竊。如果是損壞，報銷金額為：維修費−部分負擔。主要為Mastercard和高階Amex信用卡提供。

    中高階信用卡常見福利：

    退貨保障（Return Protection）
    持卡人透過信用卡購買的商品店家不提供退貨時（非常適合用於Final Sale，亦包含逾期退貨），發卡機構會理賠該商品，但發卡機構可能會要求將商品寄至發卡機構的退貨中心。

    旅遊險（Trip Protection）
    持卡人透過信用卡支付交通工具，賠付因交通工具延遲造成的額外支出（住宿/交通/餐飲及必需品）或因為不可抗力因素中斷旅遊行程，賠付已支付無法取消的旅遊費用。

    高階會籍
    持卡人可獲得免費的高階租車會籍、飯店會籍，享受租車、飯店升等等福利。持卡人亦可透過刷卡進一步晉升會籍。

    機場貴賓室
    持卡人搭乘航班時，可以免費使用機場貴賓室，並且可以免費攜帶同行者進入（通常上限為2人）。

    D. 信用紀錄累積 #

    信用卡是最簡單累積信用紀錄的方法。即使沒有SSN，沒有SSN累積的信用紀錄會透過姓名、生日等個人資訊保留。信用紀錄會影響車貸、房貸利率、保險費率等，有時候租房也會被要求提供信用紀錄。沒有信用紀錄通常比信用紀錄差更不被接納。信用紀錄累積詳情參考信用紀錄章節。

    美國常見的收單機構 #

    • Visa：信用卡Costco只接受Visa，Debit Card則不限。
    • Mastercard：通常提供手機保險。
    • Discover：同時兼具發卡銀行、收單機構，在美國基本上接受率略低於Visa、Mastercard，高於Amex。Discover有與JCB、銀聯合作，所以在日本、中國有更高接受率。
    • Amex：同時兼具發卡銀行、收單機構。雖然在美國接受率最低，但其服務、回饋、優惠活動相當有競爭力。

    熱門信用卡 #

    除了靠美國運通的國際轉卡服務，大多數的熱門信用卡都是需要信用紀錄（通常SSN都是必要的）。這些熱門信用卡通常提供較好的開卡獎勵、刷卡回饋與福利，也因此部分信用卡有年費。在申請前可以斟酌以下幾個因素來挑選適合自己消費習慣的信用卡：

    • 開卡獎勵：通常開卡提供的獎勵會遠超年費。
    • 刷卡回饋：每張信用卡都有特定類別提供加成的刷卡回饋。以學生而言，超市、餐廳是最大宗的消費，因此高回饋的超市、餐廳信用卡通常較為受歡迎。
    • 持卡福利：由於零信用紀錄新手卡中的BoA Cash Rewards與Discover it都沒有租車保險福利，因此有租車需求的可持有至少一張提供租車保險的信用卡。

    Discover信用卡 #

    雖然大部分Discover信用卡需要一定的信用紀錄才能申請，但Discover it Student Cash Back是一個不用信用紀錄又提高高額回饋的信用卡。

    Discover it Student Cash Back #

    • 開卡獎勵：$50（限推薦連結申請）。
    • 刷卡回饋：現金回饋，季度限定消費5%回饋，其他1%。第一年消費所有回饋x2。無國際手續費。
    • 福利：每年送$20（GPA > 3.0），未提供其他消費型福利。
    • 年費：免年費。
    國際交易手續費
    國際交易手續費（Foreign Transaction Fee, FTF），美國外的信用卡交易手續費，約消費金額3%。

    Amex信用卡 #

    一般而言，Amex信用卡大約擁有3個月以上的信用紀錄即可以嘗試申請。所有卡都有Amex offer、Refer a friend與開卡獎勵。惟須注意，Amex信用卡的開卡獎勵一種卡片一輩子僅能拿到一次，因此建議等到有不錯的開卡獎勵再申請。

    Amex Blue Cash Everyday #

    • 開卡獎勵：$150~$300。
    • 刷卡回饋：現金回饋，超市3%、加油2%、其他1%。
    • 年費：免年費。

    Amex Blue Cash Preferred #

    • 開卡獎勵：$200~$400。
    • 刷卡回饋：現金回饋，超市、串流影音訂閱6%、加油與交通3%、其他1%。
    • 福利：提供退貨保障。
    • 年費：$95，首年免年費。

    Amex Gold #

    • 開卡獎勵：50,000~75,000MR點。
    • 刷卡回饋：點數回饋卡（MR點），超市、全球餐廳4x、官網機票3x、其他1x。
    • 福利：無國際手續費、每個月$10 Uber Credit（可用於Uber Eats）、每個月$10 Selected Dining Credit (Grubhub, Shake & Shack, Boxed等)，一年共$240。旅遊不便險。
    • 年費：$250。

    消費類別
    Amex的消費類別定義參照連結。

    Amex offer
    特定商家現金或點金回饋，最多可達100%。類似BankAmeriDeals或Chase offer，但該系統提供更多元、優質的優惠。


    Membership Rewards (MR)
    Amex專用的銀行點數，各個卡上的MR點數會自動存到同一個帳戶。只要保留一張以上的MR卡，點數就不會過期。MR點的主要有兩種使用方法：

    • 轉入中高階卡後可轉入航空或哩程計畫。
    • 在持有AmEx Platinum for Schwab的情況下，可以把MR點已1.25 cents/point轉成現金。

    Chase信用卡 #

    除了學生卡，Chase信用卡大多需要1年以上的信用紀錄才能申請。申請通過Chase信用卡後，3~6個月內再次申請有很大的機率被拒絕。此外，24月內申請通過5張任意信用卡、不限任何銀行，Chase會直接拒絕新卡申請，即是俗稱的「5/24條款」。所有卡都有Chase offer、Refer a friend與開卡獎勵。

    Chase Freedom Flex #

    • 開卡獎勵：20,000UR點。
    • 刷卡回饋：UR點，每個季度特定類別5x、Lyft 5x、餐廳與藥局3x、其他1x。
    • 福利：手機保險（涵蓋theft、damage，須付電信費）。
    • 年費：免年費。

    Chase Sapphire Preferred #

    • 開卡獎勵：60,000~80,000UR點+$50。
    • 刷卡回饋：UR點，Lyft 5x、全球旅遊、全球餐廳2x、其他1x。
    • 福利：無國際手續費、Primary租車保險、DoorDash會員。
    • 年費：$95。
    Primary租車保險
    如果自身有車險的話，如果信用卡提供的租車保險為Secondary，當發生事故時，必須優先使用自身車險因而較有漲保費的風險。相對地，如果使用信用卡提供的租車保險為Primary，則可以優先使用信用卡保險。詳情參照租車章節。

    押金信用卡 #

    押金卡（Secured Credit Card）是一種特殊信用卡，透過事先在銀行存入一定保證金（押金），銀行依照押金額度提供信用額度，使用一段時間後銀行可能會按信用狀況退回押金。雖然也具有累積信用紀錄的功能，但通常回饋、福利、額度比不上信用卡。主要是提供給信用紀錄差的人，修復信用紀錄。

    無信用紀錄的人建議先申請零信用紀錄新手卡，不建議申請押金卡。

    常見問題 #

    如果遇到信用卡爭議款應如何處理？ 交易糾紛第一步為與商家交涉，如果交涉無效可向銀行提出爭議款。

    我拿到SSN了，請問我需要做什麼以及我的信用記錄從什麼時候開始算？ 對於留學生而言，信用紀錄是從第一張信用卡使用開始，與有沒有SSN沒有直接關係。為了讓信用局能準確找到你的資料，記得向你的信用卡發卡銀行以及支票帳戶銀行更新SSN。

    申請信用卡時，年收入該填多少？

    基本上留學生可填I-20上的金額作為收入的最低金額，但建議同時照時填寫non-taxable income（即扣掉stipend或其他薪資收入）。如果覺得父母給的錢較多或有其他獎學金收入，可以多填些。
    通常在填寫收入空格的旁邊都會有小提示，下圖是Chase信用卡申請的說明：收入包含薪資收入、股票收入以及其他人經常性給予。因此，留學生可以把父母的錢當成收入的一部份。


    如果我申請被拒了怎麼辦？ 被拒絕的第一時間可以聯絡銀行詢問原因，如果是少了資料、剛拿到SSN查不到過去信用紀錄或其他因素，可跟銀行爭取重新考慮（reconsideration），而非直接重新申請。

    如果不想繼續持有年費卡該怎麼處理？ 持有1年後可以選擇關卡或降級。降級可以保有信用歷史長度。惟須注意，降級後可能影響開卡獎勵資格。

    手上信用卡數量該多少比較好？ 如果管理的來，信用卡數量是多多益善。信用分數其中一項是帳戶數量，數量越多越高，詳情參照信用紀錄章節。但如果有預計申請Chase信用卡，須注意5/24條款。
