    前言 #

    歡迎來到匹茲堡，成為我們其中的一員。 匹茲堡台灣學人學生會創立於2008年，在1970~90年代華人學生會之後第一個致力於幫助匹茲堡地區臺灣學人、學者與學生的社團。 近年每年服務的人數約200人，是目前賓州西部規模最大的台灣學生社團，主要但不限於服務Carnegie Mellon University （CMU）和 University of Pittsburgh（Pitt）學子。

    匹茲堡生存手冊欲提供最即時、最合宜的資訊，幫助在匹茲堡地區台灣學人適應新的生活環境。 該生存手冊囊括通用的美國行前準備至匹茲堡當地生活資訊，其中參考了許多公開資訊、私人經驗以及其他學人的著作，如PITT CSSA的2016版新生生存手冊等。 作者（群）夜以繼日的蒐集資料與考究，然而即便致力追求撰寫當下內容的正確性，但隨時間推移，有些資訊可能不合時宜而須更新。 如果有錯誤、建議或需要補充的地方，煩請通知當屆兩校會長，相關人士會盡量持續更新。唯有互助的精神，才能讓這份文件持續活化。 文件中所有提到的各網站、商店、商標、產品之版權均屬原擁有者所有。 內文提到的各網站、商店均與本文作者（群）無關，本文僅提供資訊做為參考，無任何建議或廣告之意圖。 所有文件的全部或部分內容嚴禁盜為己有或在未經同意之下供作商用。
