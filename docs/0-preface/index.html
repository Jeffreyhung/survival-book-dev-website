    前言 #

    匹茲堡生存手冊欲提供最即時、最合宜的資訊， 幫助在匹茲堡地區台灣學人適應新的生活環境。 該生存手冊囊括通用的美國行前準備至匹茲堡當地生活資訊， 其中參考了許多公開資訊、私人經驗以及其他學人的著作， 如PITT CSSA的2016版新生生存手冊與UCSD TGSA新生手冊等。 部分資訊可能隨時間推移不合時宜。 如果有錯誤、建議或需要補充的地方， 歡迎利用意見回饋告訴我們你的想法， 我們將會持續更新。

    文件中所有提到的各網站、商店、商標、產品之版權均屬原擁有者所有。 所有文件的全部或部分內容嚴禁盜為己有或在未經同意之下供作商用。
