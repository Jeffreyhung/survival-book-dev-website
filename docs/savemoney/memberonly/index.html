    商家會員 #

    最後更新日期2021.05.01


    加入商家會員通常有機會獲得以下福利：

    • 首購折扣：第一次購買有折價。
    • 專屬折扣：加入會員後有便宜的折價券。
    • 免運：加入會員即可享受免運。
    • 集點換禮
    • 生日禮：不同商家可以領到什麼生日領可以參考北美省錢快報文章。
    不要交出SSN
    有些店員會向你「索取SSN」，表示加入會員會有額外折扣。警覺、警覺！這是一種推銷Store credit card術語！Gap、Banana Republic集團特別常見。
