    Referral Program #

    最後更新日期2021.05.01

    Referral Program一種不需要自掏腰包的感謝以及增進彼此感情方法。通常是個互利的，但僅是Referral方額外獲得獎勵。

    Referral Program類型 #

    美國各種消費相關產業都可能提供Referral program，包含：

    • Member Only
    • Shopping Portal
    • 信用卡
    • 銀行帳戶
    • 手機電信
    • 租屋
    • …

    提供的回饋可以為：

    • 現金
    • 點數
    • 購物金
    • 折價券
    • …

    Amex Referral Program #

    AmEx 自家信用卡（非聯名卡）可以Refer大部分信用卡，步驟如下：

    • 點選連結或將連結複製於瀏覽器的無痕模式。
    • 點選左上角View all Cards with a Referral offer
    • 點選All Personal Cards即可看到各種卡。
    • 連結提供者會在原本卡得到固定獎勵，連結使用者會依照選取卡種得到開卡獎勵。

    惟須注意，在瀏覽器無痕模式使用推薦連結有時會得到較好的開卡獎勵。
