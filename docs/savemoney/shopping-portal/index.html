    Shopping Portal #

    最後更新日期2021.05.01

    Shopping Portal的機制就是使用回饋渠道的連結進入購物網站消費，回饋渠道將賺得一部分廣告傭金返還給用戶。最重要的是，消費回饋通常可與任何消費折扣、信用卡優惠並用。

    由於有許多消費回饋渠道，同一個商家有不同的回饋比例而且隨時間變化。使用Cashback Monitor可以即時查詢最佳的回饋網站。

    常見線上購物回饋網站 #

    Rakuten #


    • Rakuten的前身為Ebates，超過20年歷史、用戶最多且操作方便。
    • 可使用官網、網頁瀏覽器插件、或者手機app三種方式進行操作。
    • 少數支援實體店購物的回饋網站（需手動加入）。
    • 與Amex合作，如果持有MR點的信用卡，可以將所有現金回饋以1MR : 1cent比例轉換成MR點數。
    • 每季結算。
    • 領出回饋的方式主要以支票、Paypal或轉入Amex MR帳戶，需要超過$5才能領出。

    TopCashback #


    • 超過15年歷史。
    • 可使用官網、網頁瀏覽器插件、或者手機app三種方式進行操作。
    • 商家比例覆蓋廣。
    • 經常有自家額外金額回饋的促銷活動（例如：滿$10送$2.5）。
    • 回饋比例通常最高。
    • 每筆交易單獨結算（4-14週）。
    • 領出回饋的方式較為多樣（ACH、Paypal、Gift Card等），沒有最低額限制，現金以外獲得最多額外5%回饋。

    RebateKey #


    • 主要是讓商家（主要在Amazon）提供高額現金回饋以快速打出新產品的知名度。
    • 回饋金額可高達100%。
    • 完成消費後需要將訂單代號填寫至RebateKey網站。
    • 30天後結算。
    • 領出回饋的方式主要以支票或Paypal。
    • 產品侷限。

    航空公司 #


    • 許多航空公司（例如美國航空、阿拉斯加航空、西南航空等）都有自己的Shopping portal，提供哩程回饋。
    • 定期有額外哩程回饋的促銷活動（例如：滿$125送500mile）。
    • 新哩程獲得可防止航空哩程計畫過期。

    銀行 #


    • 主要是Chase、Citi等提供銀行點數回饋。

    常見實體店購物回饋 #

    Pei #


    • 將信用卡綁定在App後，以後在特定的實體店消費後都會自動獲得消費回饋。
    • 沒有連接卡的限制，盡可能將所有卡榜定在上面。
    • 採用Plaid公司的技術，是被銀行app採用的公司。
    • 領出回饋的方式為Paypal、比特幣以及Gift Card，需要超過$25才能領出。

    Dosh #


    • 將信用卡綁定在App後，以後在特定的實體店消費後都會自動獲得消費回饋。
    • 合作商家侷限，但具有特色，例如Uber Eats、Sam’s club等。
    • 採用Plaid公司的技術，是被銀行app採用的公司。
    • 領出回饋的方式為Paypal、Venmo、ACH，需要超過$25才能領出。
